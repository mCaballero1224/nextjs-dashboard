## Next.js App Router Course

This is a dashboard built using the Next.js App Router Course. I'm a little rusty at Next.js, so I figured I would use this to brush up on some concepts as well as pick up any changes I might've missed since the last time I did web development.

For more information, see the [course curriculum](https://nextjs.org/learn) on the Next.js Website.
